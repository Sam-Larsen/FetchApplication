PointSpender

Fetch Application Project
Reads in a file and spends a specified number of points from oldest to newest.

To run:
Ensure PointSpender.java, Time.java, Transaction.java and file containing transactions are located in the same directory
Compile in terminal using the command: javac PointSpender.java
Run using the command: java PointSpender <Int: Points to Spend> <String: File Name>
in the example the run command would be: java PointSpender 5000 transactions.csv
